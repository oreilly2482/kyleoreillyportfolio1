Automatic Egg Cracker


Finalist Design for InventOregon Prototyping competition. This was designed to be an improvement upon current assitive devices for cracking eggs.
This system loads an egg, cracks it, ejects the shell, and load another egg. This is all triggered by a limit switch on the top of the device, so it only operates if eggs are in place.

Electronics Design:

  • Controlled by Arduino
  • DC motor to drive cracking arms
  • Servo motor to control egg dispensing
  • Limit switches for arm homing, and system triggering


Mechanical Design:

  • A Lead screw drives the arms- this component is removable for easy cleaning.
  • Designed so arms cannot run off of the lead screw, meaning only one limit switch was required for positioning.
  • 3D printed and designed in Solidworks and 3D Printed using PLA.
